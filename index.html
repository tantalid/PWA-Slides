Разные платформы

Progressive Web Application (PWA)

  • Это сайт
  • Устанавливается
  • Выглядит почти как нативное приложение
  • Способен работать в оффлайн
  • Способен присылать пуш-уведомления

Какие плюшки от PWA?

  • Простота в разработке
  • Одно приложение для всех платформ (+ Windows Store)
  • Маленький вес
  • Решен вопрос с SEO

Для бизнеса

  • Кроссплатформенность
  • Одна команда на весь проект
  • Установку предлагает браузер
  • Моментальная установка
  • Пользователю не нужно обновлять

Про технологии

Прогрессивное приложение подразумевает

  • Отзывчивый / адаптивный дизайн
  • Веб манифест
  • Автономный режим
  • Только HTTPS
  • Веб API, такие как пуш-уведомления

Веб манифест

			{
				"lang": "ru",
				"name": "Progressive Web Application",
				"short_name": "PWA Demo",
				"description": "Это тестовое Progressive Web Application",
				"icons": [{
					"src": "./img/icon/app-icon.png",
					"sizes": "64x64",
					"type": "image/png"
					}, ..., {
					"src": "./img/icon/app-icon-256.png",
					"sizes": "256x256",
					"type": "image/png"
				}],
				"start_url": "/docsite/",
				"display": "standalone", /* fullscreen, minimal-ui, browser */
				"orientation": "portrait-primary"
			}

Подключение

			<link rel="manifest" href="manifest.json">

Итоги

Достоинства PWA

  • Простота в разработке
  • Кроссплатформенность
  • Моментальная установка
  • Решен вопрос с SEO

Недостатки PWA

  • Это не серебряная пуля, не покрывает все задачи
  • Свежий подход
  • Нет поддержки сервис-воркеров в Safari (AppCache)
